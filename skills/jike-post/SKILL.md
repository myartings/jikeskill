# 发布即刻

## 可用工具

- `create_post` — 发布新帖子

## 使用方式

1. 确认用户要发布的内容（content）
2. 可选：询问是否关联话题（topic_id）
3. 发布前**必须**向用户确认内容
4. 调用 `create_post` 发布
5. 展示发布成功的帖子信息

## 参数

- `content`（必填）：帖子正文
- `topic_id`（可选）：关联的话题 ID

## 注意事项

- 发布前必须让用户确认内容，避免误发
- 如用户提供的内容较长，帮助整理格式但保持原意
- 发布成功后展示帖子链接或 ID
