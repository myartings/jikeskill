# 即刻登录

## 前置条件

MCP 服务器必须在运行。如未运行，先执行：
```shell
cd ~/.openclaw/skills/jike && bash scripts/setup.sh
```

## 登录流程

1. 调用 `check_login_status` 检查是否已登录
2. 如已登录，告知用户当前登录的账号信息
3. 如未登录，执行以下步骤：
   a. 调用 `get_login_qrcode` 获取二维码图片，记下返回的 `uuid`
   b. 将二维码展示给用户，提示用户打开即刻 App 扫描
   c. **【关键】展示二维码后，必须立即调用 `wait_for_login`，传入上一步的 `uuid`。不要等用户回复，不要等用户说"已扫码"，展示二维码的同一轮回复中就要调用 `wait_for_login`。** 此接口会在服务端轮询最多 180 秒，扫码确认后自动返回。
   d. 成功后告知用户已登录

⚠️ **常见错误：展示二维码后停下来等用户回复，导致 `wait_for_login` 从未被调用，登录永远无法完成。务必在展示二维码后立即调用。**

## 登出

调用 `logout` 清除本地保存的 token。

## 技术说明

- 登录不需要公网 IP，不需要回调。`wait_for_login` 是客户端主动轮询即刻服务器，纯出站请求。
- 二维码有效期约 3 分钟，超时需重新获取
- Token 保存在本地文件中，下次启动无需重新登录
- 如遇到 401 错误，token 会自动刷新；刷新失败需重新扫码
