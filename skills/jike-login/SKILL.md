# 即刻登录

## 登录流程

1. 调用 `check_login_status` 检查是否已登录
2. 如已登录，告知用户当前登录的账号信息
3. 如未登录，执行以下步骤：
   a. 调用 `get_login_qrcode` 获取二维码图片
   b. 将二维码展示给用户，提示用户打开即刻 App 扫描
   c. 记下返回的 `uuid`
   d. 调用 `wait_for_login`，传入 `uuid` 参数
   e. 等待扫码确认（最长 180 秒）
   f. 成功后告知用户已登录

## 登出

调用 `logout` 清除本地保存的 token。

## 注意事项

- 二维码有效期约 3 分钟，超时需重新获取
- Token 保存在本地文件中，下次启动无需重新登录
- 如遇到 401 错误，token 会自动刷新；刷新失败需重新扫码
